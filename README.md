# Triscplay  
<========================>  
CESI TON SCORE !!!!  
<========================>  
Triscplay est une plateforme communautaire permettant le partage de scores et de reccords sur tout vos jeux.   
  
Ce projet est réalisé en étroite collaboration pour l'association des troubadours de la robotique et de l'informatique situés au coeur du Léon (TRISCL (d'ou le nom))
